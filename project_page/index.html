Abstract

We propose a framework where Computer-Use Agents (CUA) act as judges to assist Coders in automatic GUI design. We introduce AUI-Gym, a benchmark of 52 applications with 1560 tasks. In our Coder-CUA Collaboration, the CUA evaluates interface functionality and solvability to guide the Coder's design refinements. We also provide a CUA Dashboard to visualize navigation histories for iterative improvement. This approach shifts interface design toward agent-native efficiency and reliability. Our code and dataset are available at https://github.com/showlab/AUI.

What is AUI?

Illustration of Humans Collaboration vs. our Coder-CUA Collaboration in term of UI designs.
Left: Most GUIs are designed by humans and optimized for user experience (e.g. aesthetics), forcing trained agents to adapt to human-oriented behaviors. Right: Our Coder-CUA Collaboration framework leverages Coder as Designer and CUA as Judge together, enabling more reliable task execution and improved usability for agents.

Coder–CUA Collaboration

We propose a framework where the Coder acts as Designer, responsible for initializing and iteratively refining the UI, while the CUA serves as Judge, exploring the interface and providing feedback. This collaboration is driven by two feedback signals: Task Solvability, ensuring all features are functional, and Navigation Success, optimizing the layout for agent execution.

Qualitative Analysis

Revision from Task Solvability Feedback: Addressing missing functionalities to ensure task feasibility.
Revision from CUA Navigation Feedback: Enhancing agent accessibility via de-stylization and simplified layouts (e.g., adding buttons with clear boundaries).

CUA Dashboard

To efficiently process long agent trajectories, we introduce the CUA Dashboard. This single-image visualization compresses multi-step interactions by highlighting only key interactive regions and their temporal order. It reduces visual token redundancy by over 76% while preserving essential context, enabling the Coder to quickly identify and fix navigation bottlenecks.
