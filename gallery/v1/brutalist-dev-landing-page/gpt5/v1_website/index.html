Accessibility and Keyboard

      

        
Keyboard navigation

        
Use Tab and Shift+Tab to move between controls. Use Enter to activate a focused button or link. Arrow keys switch code tabs. Press Esc to close modals or to pause the terminal animation.

      

      

        
Reduced motion

        
If your system has reduced motion enabled, we minimize animations and render a static terminal sample. You can also manually pause the terminal animation via the "Pause Terminal" button in the header.

      

      

        
Anchor navigation offsets

        
Anchors such as "Docs", "Pricing", and "Customers" respect the sticky header. We ensure proper scroll margins so headings are not hidden behind the header.

      

    


    
    

      
FAQ

      

        
What is DevPlat?

        
DevPlat is a brutalist developer platform that prioritizes predictable APIs, honest docs, and fast tooling. We avoid visual clutter and deliver essentials with a consistent CLI and HTTP interface.

      

      

        
How do I authenticate?

        
Send an Authorization: Bearer header with your API key. You can find code examples in the "Authenticate" card above. Our SDKs wrap this for you.

      

      

        
Do you support streaming?

        
Yes. Our streaming endpoints use chunked transfer encoding for continuous delivery. Examples are in the "Stream Responses" card.

      

      

        
What about enterprise needs?

        
We provide SSO, custom terms, and higher SLAs for enterprise customers. Use the "Contact Sales" CTA in the pricing table to start a conversation.

      

      

        
Can I self-host?

        
We offer a limited self-hosted distribution for regulated environments. Contact us for details.
